# Basic

Most basic usage is one without validators. Formeus will handle updates and return
the latest values.

Initialize the form by passing your form initial values to the `initial` argument.

Use the `values` object to read the latest form values and set them to their respective fields.

<Callout type="info" emoji="ℹ️">
  This signifies that the `values` object expects a _controlled UI component_.
</Callout>
